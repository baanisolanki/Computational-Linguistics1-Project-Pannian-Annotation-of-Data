# Computational-Linguistics1-Project-Pannian-Annotation-of-Data
course project for CL1

## 1. 'manual_annotations/'
This folder contains all the human-verified, gold-standard datasets created by the team, along with their analysis.
'gold_manually_annotated_english_dataset.conll': The primary, verified manual annotation for the English corpus using the two-tier Paninian and POS tagset.
'gold_manually_annotated_hindi_dataset.conll': The primary, verified manual annotation for the Hindi corpus.
'gujarati_annotated_dataset.conll': The manual annotation for Gujarati, created with the assistance of a native speaker.
'english.conll': An intermediate or supplementary annotation file for the English dataset.
'analysis/':

'graphs.py': The Python script used to calculate the frequency of tokens/tags in the manual datasets.

'english_manual_analysis.png' & 'hindi_manual_analysis.png': Bar charts visualizing the total count of each linguistic tag (like k1, k2, V, etc.) used in the respective manual datasets.

---

## 2. 'original_datasets/'
This folder holds the raw source data, qualitative observation notes, and custom programmatic tools developed for tagging and conversion.

'english_dataset.conll' & 'hindi_dataset.conll': The original, unannotated sentences collected from popular newspapers.
'gujarati_dataset_and_gloss.conll' & 'telugu_dataset_and_gloss.conll': The raw 70 Gujarati and 50 Telugu sentences mapped with a loose meaning gloss to preserve word relations.
'observations_gujarati.txt' & 'observations_telugu.txt': Qualitative linguistic notes detailing the feasibility of applying the Paninian framework to these languages (e.g., noting that Telugu was abandoned due to its complex Dravidian morphosyntax).
'eng_data_first50_deepti_tagset.txt' & 'hindi_data_first50_deepti_tagset.txt': The first 50 sentences of English and Hindi re-annotated using Professor Deepti's validated POS mapping framework.
'python_code_to_annotate_eng.py': The rule-based Python script developed to automatically tag English sentences.
to run: do python python_code_to_annotate_eng.py 
do this in the correct directory
'output_eng_by_python_code.conll': The direct, raw output generated by the Python tagging script.

### 'original_datasets/comparison_python_our_dataset/'
Contains files evaluating the performance of the Python tagger.
'comparing_python_output_deppti_data.py': Script to compare the Python tagger's output against the gold-standard Deepti dataset.
to run: python comparing_python_output_deppti_data.py
'comparison_deepti_python_output.conll': A text file showing the side-by-side tagging results for direct comparison.
'comparison_graph.py': Script to generate visual charts of the tagger's performance.
run via: python comparision_graph.py
'confusion_matrix.png': A visual heatmap showing where the Python script misclassified tags (e.g., predicting k2 instead of k7).
'tagging_errors_final.png': A bar chart displaying the top 10 most frequent tagging mistakes made by the machine.
'tagging_analysis_report.md': A text report detailing the F1 scores and analysis of the Python tagger's performance.

### 'original_datasets/paninian_to_ud/'
'pythoncode_paninian_to_ud.py': The script developed to convert Paninian dependencies to Universal Dependencies (UD).
to run this code: 
    python paninian_to_ud.py output_eng_by_python_code.conll output_python_pannian_to_ud_eng.txt
    P.S: do this in correct directory

'output_python_pannian_to_ud_eng.txt': The resulting converted English dataset produced by the script.

## 3. 'AI_results/'
This folder contains the prompts, context files, generated outputs, and performance analysis for the three Large Language Models tested: ChatGPT, Gemini, and Gemma 2b (labeled as Ollama).

'Prompts_Code_used_for_AI_testing.txt': Contains the exact zero-shot, few-shot, and instruction-based text prompts used to query the AI models.

### 'AI_results/datafiles_used/'
Contains the specific datasets fed to the AIs as context during prompting.
'zeroshot/': 'batch1_english.conll', 'batch1_hindi.conll' (The first 50 raw sentences).
'fewshot/': 'trainingdata_english.conll', 'trainingdataset_hindi.conll' (pre-annotated examples) and 'batch2_english.conll', 'batch2_hindi.conll' (the next 50 sentences).
'instructions/': 'instructions.txt' (the rules explaining the tagset) and 'batch3_english.conll', 'batch3_hindi.conll' (the final 50 sentences).

### 'AI_results/[Model Name]/' (ChatGPT, Gemini, Ollama)
Each model directory contains an identical structure to evaluate its specific performance.

'zeroshot/', 'fewshot/', 'instructions/': These folders contain the raw '.conll' outputs generated by the AI for both English and Hindi, using both the original tagset and Deepti Ma'am's tagset. They also contain an 'analysis/' subfolder with a 'tag_comparison_english.png' to compare tag distribution.
'[model]_analysis_eng/': Contains subdirectories ('zero', 'few', 'instr') for evaluating the English outputs. Inside each subfolder:

'testing_[model]_[method].py': The Python script used to calculate accuracy, F1, precision, and recall against the gold standard.

for running: python {filename.py} 
in correct directory

'f1_...png', 'precision_...png', 'recall_...png': Bar charts visualizing the respective evaluation metrics for each tag.

'[model]_[method]_confusion_matrix.png': A heatmap showing the model's true vs. predicted classifications.

'[method]_[model].png': Aggregate summary graphs of the model's performance for that prompting method.
